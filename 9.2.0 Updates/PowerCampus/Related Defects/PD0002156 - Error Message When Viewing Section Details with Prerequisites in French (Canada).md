# PD0002156 - Error Message When Viewing Section Details with Prerequisites in French (Canada)  

## **Issue Summary**  
**Issue:**  
- When attempting to view the details of a **course section with prerequisites** in **PowerCampus Self-Service**, an error occurs **only when the browser language is set to French (Canada)**.  
- The error message displayed is:  
``` There was an error in PowerCampus Self-Service. Try again. ```

**Severity:** Moderate  
**Product Line:** PowerCampus - Self-Service  
**Resolution Date:** 2022-07-29  

---

## **Replication Steps**  
1. Set the **browser language** to **French (Canada)**.  
2. Open **PowerCampus Self-Service**.  
3. Navigate to **Course Registration > Section Search**.  
4. Select a **course section that has prerequisites**.  
5. Click on **View Details**.  
6. The system **throws an error message** instead of displaying section details.  

---

## **Workaround**  
- **Change the browser language** to **English (US)** or another supported language before accessing the section details.  
- If changing the browser language is not an option, **use PowerCampus version 9.2.0**, where the issue is resolved.  

---

## **Expected vs. Actual Result**  
| **Expected Result** | **Actual Result** |
|--------------------|------------------|
| Section details should be displayed correctly in French (Canada). | Error message appears, and section details cannot be viewed. |

---

## **Client Impact**  
- **French (Canada) users are unable to view section details** for courses with prerequisites.  
- **Frustration and usability issues** for users who rely on the **French language interface**.  
- **Inconvenience** in the registration process, requiring **manual workarounds**.  

---

## **Resolution**  
- **Fixed in PowerCampus version 9.2.0.**  
- The issue was related to **localized prerequisite validation** in the French (Canada) language setting.  
- Adjustments were made to ensure **correct handling of prerequisites in all languages**.
